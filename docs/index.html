Política de Privacidad de AhorrApp

Última actualización:10/01/2026

En AhorrApp respetamos la privacidad de los usuarios y cumplimos con la normativa vigente en materia de protección de datos, incluido el Reglamento (UE) 2016/679 (RGPD). Esta Política de Privacidad explica qué datos se recopilan, con qué finalidad y cómo se protegen.

---

1. Responsable del tratamiento

Responsable: D4vRAM
Contacto: d4vram369@gmail.com

---

2. Datos que se recopilan

AhorrApp recopila únicamente los datos necesarios para el funcionamiento de la aplicación:

  • Identificador del dispositivo (device_id), utilizado para:
    • Evitar abusos del sistema
    • Asociar datos de uso
    • Gestionar licencias o accesos
  • Datos introducidos por el usuario en la base de datos comunitaria:
    • Precios
    • Productos
    • Información asociada al registro de dichos datos
  • Datos técnicos básicos necesarios para la comunicación con el backend.

AhorrApp no recopila nombres reales, direcciones postales, datos bancarios ni información sensible.

---

3. Base de datos y backend

La aplicación utiliza una base de datos comunitaria alojada en Supabase, una plataforma open source que actúa como proveedor de backend.

Los datos se almacenan en servidores gestionados por Supabase, cumpliendo con estándares de seguridad y protección de datos.

---

4. Finalidad del tratamiento

Los datos se utilizan exclusivamente para:

  • Permitir el funcionamiento de la base de datos comunitaria
  • Registrar y mostrar precios y productos
  • Prevenir usos indebidos o abusos
  • Gestionar posibles sistemas de licencia o acceso
  • Enviar mensajes o notificaciones dentro de la aplicación, si se habilita esta función
---

5. Comunicaciones y mensajería

AhorrApp puede incluir funciones de comunicación o mensajería dentro de la aplicación. Estas comunicaciones se limitan al ámbito funcional de la app y no se utilizan con fines publicitarios.

---

6. Base legal

El tratamiento de los datos se basa en:

  • El consentimiento del usuario
  • La necesidad de ejecutar las funcionalidades principales de la aplicación
  • El interés legítimo en garantizar la seguridad y el correcto uso del servicio
---

7. Conservación de los datos

Los datos se conservan únicamente mientras sean necesarios para el funcionamiento de la aplicación. El usuario puede dejar de usar AhorrApp en cualquier momento.

---

8. Cesión de datos

Los datos no se venden ni se ceden a terceros. Solo pueden ser tratados por proveedores técnicos necesarios para el funcionamiento del servicio (como Supabase).

---

9. Derechos del usuario

El usuario puede ejercer los siguientes derechos:

  • Acceso
  • Rectificación
  • Supresión
  • Limitación del tratamiento
  • Oposición

Para ejercer estos derechos, puede contactar a través de:

10. Seguridad

Se aplican medidas técnicas y organizativas razonables para proteger los datos frente a accesos no autorizados, pérdidas o usos indebidos.

---

11. Cambios en la política

Esta Política de Privacidad puede actualizarse para reflejar cambios en la aplicación o en la normativa. Las modificaciones se publicarán en esta misma página.
